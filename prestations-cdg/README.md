# Modèle Gestion des prestations de CDG pour Grist 

Ce modèle Grist permet de recenser les souscriptions des collectivités (communes, agences, CCAS...) aux prestations fournies par les CDG dont leur territoire dépend. Il permet également de générer des courriers à envoyer par voie postale en fonction de la prestation concernée.

Pour l'utiliser, téléchargez le fichier `Gestion des prestations de CDG.grist`, puis dans votre instance Grist, ouvrez le dossier de votre choix, cliquez sur **Nouveau** > **Importer un document**, et sélectionnez le fichier précédemment téléchargé.

Vous pouvez également essayer le modèle sur l'instance de Grist Labs (nécessite un compte) :
https://docs.getgrist.com/dgtMPMdaZBPX/Modele-gestion-des-prestations-de-CDG-11
