# Modèle Tickets d'assistance pour Grist

Ce modèle permet de recueillir des demandes sous forme de tickets, de partager leur statut avec leurs auteurices, et de les notifier automatiquement de l'évolution de leurs demandes à l'aide d'un workflow n8n.

Pour l'utiliser, téléchargez le fichier `Modèle tickets d'assistance.grist`, puis dans votre instance Grist, ouvrez le dossier de votre choix, cliquez sur **Nouveau** > **Importer un document**, et sélectionnez le fichier précédemment téléchargé.

Pour utiliser le workflow n8n, téléchargez le fichier `Tickets d'assistance Grist.json`, puis puis dans votre instance n8n, clkiquez sur **...** > **Import from file...**, et sélectionnez le fichier précédemment téléchargé. Lisez bien les instructions présentes dans le document Grist pour savoir comment connecter ce dernier au workflow n8n.

Vous pouvez également essayer le modèle Grist (sans le workflow n8n) sur l'instance de Grist Labs (nécessite un compte) :
